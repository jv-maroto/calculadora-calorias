  OPCIÓN 2: MÉTODO MANUAL (Línea de comandos)
═══════════════════════════════════════════════════════════════

1️⃣ Abrir CMD en esta carpeta:
   C:\xampp\htdocs\Calculator

2️⃣ Ejecutar comandos:

   git remote add origin https://github.com/TU_USUARIO/TU_REPO.git
   git push -u origin master

   (Reemplaza TU_USUARIO y TU_REPO con los tuyos)

3️⃣ Si te pide contraseña:
   - Usuario: tu usuario de GitHub
   - Password: Personal Access Token (NO tu contraseña)

   Crear token en: https://github.com/settings/tokens
   - Generate new token (classic)
   - Marca solo "repo"
   - Copia el token y úsalo como contraseña

═══════════════════════════════════════════════════════════════
  PROBLEMAS COMUNES
═══════════════════════════════════════════════════════════════

❌ "Permission denied (publickey)"
   → Usa HTTPS, no SSH
   → URL correcta: https://github.com/usuario/repo.git

❌ "Authentication failed"
   → Usa Personal Access Token, no contraseña
   → Crear en: https://github.com/settings/tokens

❌ "Repository not found"
   → Verifica que creaste el repositorio en GitHub primero
   → Verifica el nombre del usuario y repo

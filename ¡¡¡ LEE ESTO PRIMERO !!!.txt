┌───────────────────────────────────────────────────────────────┐
│  OPCIÓN 2: COMANDOS MANUALES (Para usuarios avanzados)       │
└───────────────────────────────────────────────────────────────┘

    PASO A - Crear repositorio:
    ───────────────────────────────────────────────────────────
    1. Ve a: https://github.com/new
    2. Nombre: calculadora-calorias
    3. NO marcar nada
    4. Click "Create repository"

    PASO B - Abrir CMD aquí:
    ───────────────────────────────────────────────────────────
    1. Abre CMD en esta carpeta (Shift + Click derecho → "Abrir ventana de comandos aquí")
    2. O escribe: cd C:\xampp\htdocs\Calculator

    PASO C - Ejecutar comandos:
    ───────────────────────────────────────────────────────────
    Copia y pega (reemplaza TU_USUARIO con tu usuario de GitHub):

    git remote add origin https://github.com/TU_USUARIO/calculadora-calorias.git
    git push -u origin master


┌───────────────────────────────────────────────────────────────┐
│  ¿QUÉ PASA SI ME PIDE CONTRASEÑA?                           │
└───────────────────────────────────────────────────────────────┘

    ⚠️  NO uses tu contraseña normal de GitHub

    ✅  Usa un "Personal Access Token":

    1. Ve a: https://github.com/settings/tokens
    2. Click "Generate new token (classic)"
    3. Nombre: "Calculadora Calorias"
    4. Marca solo: repo
    5. Click "Generate token"
    6. COPIA el token (empieza con ghp_)
    7. Pégalo como contraseña en CMD
